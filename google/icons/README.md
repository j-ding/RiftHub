# Extension Icons

This extension requires three icon sizes:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Creating Icons

You can create simple icons using:

1. **Online Tools**:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. **Design Software**:
   - Photoshop, GIMP, Figma, etc.

3. **Quick SVG to PNG**:
   - Create an SVG with a game controller or Riftbound logo
   - Convert to PNG at required sizes
